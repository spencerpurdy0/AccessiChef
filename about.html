About AccessiChef

Our Mission

AccessiChef is designed to make cooking accessible to everyone, regardless of ability. Our application provides recipe instructions in multiple formats to accommodate users with various disabilities, including vision impairments, motor limitations, and cognitive challenges.

Accessibility Features

  • Screen Reader Compatibility: All elements are properly labeled with ARIA attributes for screen reader navigation.
  • Keyboard Navigation: The entire application can be operated using only a keyboard.
  • High Contrast Mode: Optional high contrast theme for users with vision impairments.
  • Text Size Adjustment: Easily increase or decrease text size without affecting layout.
  • Step-by-Step Instructions: Recipes are broken down into clear, manageable steps.
  • Timer Functionality: Visual and audio timers for cooking steps.
  • Simplified View: Reduced-distraction view for users with cognitive disabilities.

About the Developer

This application was developed as a semester project for COMP 5970/6970 Assistive and Accessible Computing at Auburn University. The goal was to create a practical, accessible mobile web application that demonstrates universal design principles and provides a valuable service to users with disabilities.

Accessibility Statement

AccessiChef is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

The application follows the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

If you encounter any accessibility barriers while using AccessiChef, please contact us.
